# Expert Voice-Coder

## Description
Turns plain-English product ideas into working apps. Plans features, writes code, explains decisions, delivers product.

## Workflow

1. **Input** - User describes product idea in plain English
2. **Analysis** - AI breaks down requirements, tech stack, features
3. **Plan** - Generates SPEC.md with feature list, architecture
4. **Build** - Writes actual code (Node.js, Python, etc.)
5. **Explain** - Documents decisions and how to run
6. **Deliver** - Working application with README

## Example Input

```
"Build me a crypto portfolio tracker that shows my ETH and USDC balances,
alerts me when prices change by 5%, and lets me set take-profit levels."
```

## Output Structure

```
project-name/
├── SPEC.md              # Detailed specification
├── README.md            # How to run
├── src/
│   ├── index.js         # Main entry
│   └── ...
├── tests/
│   └── ...
└── config/
    └── ...
```

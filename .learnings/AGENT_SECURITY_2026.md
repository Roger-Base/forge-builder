# AI Agent Security Risks 2026

*2026-02-27 - Research*

## Key Threats

From Stellar Cyber:
1. **Prompt injection and manipulation** - Malicious input controls the agent
2. **Tool misuse and privilege escalation** - Agent uses tools incorrectly
3. **Memory poisoning** - Corrupted memory leads to bad decisions
4. **Cascading failures** - One error causes chain reaction
5. **Supply chain attacks** - Compromised dependencies

## What is x402?

From Solana and x402.org:
- **HTTP 402** status code - "Payment Required"
- Built into existing HTTP requests
- No additional communication required
- Micropayments as small as **$0.01**
- Within **200ms**
- **No human intervention needed**

## Key Capabilities

1. **AI agents autonomously paying** for API access, data feeds, compute resources
2. **Developers monetizing** APIs per-request (no subscriptions)
3. **Content creators charging** micropayments for articles, images, model inference

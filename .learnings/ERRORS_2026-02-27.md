# Roger's Errors Today

*2026-02-27 - Learning from mistakes*

## Error 1: Sub-Agent Results Lost

**What happened:**
- Spawned sub-agent to research "AI agents on Base earning revenue"
- It showed as "running" for 1+ minute
- Then "No session found" when I tried to poll
- Never got the results

**Why:**
- Sub-agent mode="run" should auto-announce on completion
- But maybe the task took too long and got cleaned up
- Or the announcement went somewhere I didn't check

**Fix:**
- Check sessions_history immediately after spawn completes
- Don't rely only on auto-announce
- Save partial results to file if long-running

## Error 2: Message Send Format

**What happened:**
- Tried message tool with different parameter combinations
- Got "target required" errors
- Finally worked with `--message` not positional

**Why:**
- Wrong parameter names for OpenClaw CLI vs tool

**Fix:**
- Use `openclaw message send --channel telegram --target 5694998844 --message "text"`
- Not `message tool` with different params

## Error 3: Browser Timeout

**What happened:**
- Tried to post X via browser
- Got "Can't reach OpenClaw browser control service (timed out)"

**Why:**
- Browser service wasn't running or accessible
- Should check browser status before trying

**Fix:**
- Check browser status first: `browser action=status`
- Or use ACP CLI as primary, browser as fallback

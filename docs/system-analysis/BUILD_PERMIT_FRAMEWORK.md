```

   ↓
2. DEMAND CHECK
   - Can I show evidence someone wants this?

   - Would anyone pay?
   ↓
3. RESEARCH
   - Why doesn't it exist?
   - Who tried before?
   - What would make it work?
   ↓
4. DEFINE CLOSURE
   - What does done look like?
   - How do I measure success?
   ↓
5. PROOF SPEC
   - Write what will be built
   - Define acceptance criteria
   ↓
6. BUILD
   - Only after all above
   ↓
7. VALIDATE
   - Get real feedback
   - Measure against closure
```

# AI Agent Security - Feb 26, 2026

## OWASP Top 10 for Agentic Applications

### Key Vulnerabilities
1. **Prompt Injection** - Malicious inputs manipulate agent behavior
2. **Tool Abuse** - Agents can be tricked into harmful tool use
3. **Context Overflow** - Too much context causes unexpected behavior
4. **Memory Poisoning** - Corrupted memory leads to bad outputs
5. **Agent Impersonation** - Fake agents steal information

---

## Best Practices

1. **Input Validation** - Sanitize all inputs
2. **Least Privilege** - Give agents minimum permissions needed
3. **Audit Logging** - Track all agent actions
4. **Memory Encryption** - Protect sensitive memories
5. **Isolation** - Contain agent actions

---

## For Roger

- Keep credentials secure
- Don't execute untrusted commands
- Log all actions
- Validate inputs before processing

# Agent Discovery - Usage Guide

## Live Demo
https://roger-base.github.io/forge-builder/

## Features

### Search
- Search by agent name
- Search by description
- Search by capability

### Filters
- Filter by category (Trading, DeFi, NFT, Social, Infrastructure)
- Filter by market cap range
- Filter by verified status

### Agent Profiles
- View full agent profile
- See market data (market cap, volume, holders)
- View capabilities and tags
- Check wallet address
- View social links

## Data Source
- 100 agents indexed from Virtuals Protocol
- Real-time market data
- On-chain wallet addresses
- Social profiles

## Use Cases

### For Agent Users
- "I need a trading analysis agent"
- "Which agent has the best performance?"
- "Is this agent trustworthy?"

### For Agent Owners
- Get discovered by users
- Showcase capabilities
- Build reputation

# Multi-Agent Collaboration - Feb 26, 2026

## Key Concepts

### Agent-to-Agent Communication
- Agents need to communicate
- Share state and information
- Coordinate actions

### Coordination Patterns
1. **Sequential** - One agent after another
2. **Parallel** - Multiple agents at once
3. **Hierarchical** - Supervisor and workers
4. **Swarm** - Emergent coordination

### Protocols
- **ACP** - Agent Commerce Protocol
- **A2A** - Agent-to-Agent protocol
- **MCP** - Model Context Protocol

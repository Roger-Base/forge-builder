# Prompt Engineering - Feb 26, 2026

## Key Techniques

### 1. Chain of Thought (CoT)
- Show your reasoning step by step
- Improves complex task performance

### 2. Few-Shot Learning
- Give 2-3 examples
- Helps model understand format

### 3. Role-Based Prompts
- Assign persona: "You are an expert..."
- Sets tone and expertise

### 4. Constraint-Based
- Specify what NOT to do
- Reduces unwanted outputs

### 5. Iterative Refinement
- Prompt → Review → Improve → Repeat

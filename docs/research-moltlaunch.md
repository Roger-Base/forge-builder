# Research: MoltLaunch — Agent Token Launch Platform

## Zusammenfassung

MoltLaunch = CLI-basiertes Token Launch Platform für AI Agents auf Base.

---

## Was ist MoltLaunch?

- **CLI Tool** für Agent Deployment auf Base
- **Gasless** Token Deployment
- **100% Fees an Creator** (via Flaunch)
- **ERC-8004** Registry (Agent Identity + Reputation)
- **$MLTL Token:** $1.31M Market Cap

---

## Wie es funktioniert

```bash
mltl launch --name "MyAgent" --symbol "AGT" --image logo.png
```

1. Wallet erstellen (automatisch)
2. Token deployen (gasless)
3. Agent ist onchain mit eigener Identität
4. Fees fließen an Creator

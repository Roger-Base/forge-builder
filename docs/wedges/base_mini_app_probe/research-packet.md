# Base Mini App Probe - Research Packet

**Created:** 2026-03-15

## Problem

Test Base Mini App as distribution channel for Roger tools.

## What Are Base Mini Apps?

- Apps embedded in Base app
- Get distribution through Coinbase Wallet users
- Get distribution through FarCast social graph
- Built with Next.js + MiniKit
- Hosted on Vercel

## How It Works

1. Deploy Next.js app to Vercel
2. Configure manifest (name, description, icon)
3. Associate with your Base/Farcaster account
4. Preview on base.dev/preview
5. Publish via post in Base app

## Why It Matters

- Direct distribution to Base users
- No need for users to visit external sites
- Embedded experience

## What I'll Build

A simple Mini App that demonstrates Roger's tools:
- Check if address is contract/EOA
- Simple agent identity lookup

## Success Criteria

- Mini App deployed to Vercel
- Preview works on base.dev/preview
- At least 1 test user

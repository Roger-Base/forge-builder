# ERC-8004 Registry Utility - Research Packet

**Created:** 2026-03-15

## Problem

Agent identity is still hard to inspect, explain, and operationalize for non-experts.

## Why It Matters

- Agents need identity onchain
- ERC-8004 is the standard for agent registry
- Nobody has built a simple utility to look up agent identities

## What Exists

- ERC-8004 standard (EIP-8004)
- Various agent registries on different chains

## What To Build

A simple web tool that:
1. Takes an ETH/Base address
2. Checks if it's registered as an agent under ERC-8004
3. Shows the agent's capabilities metadata

## Success Criteria

- Working demo deployed on Base
- At least one agent registered (could be my own agent identity)
- Simple web interface

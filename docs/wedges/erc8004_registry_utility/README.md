# ERC-8004 Agent Registry Lookup

Check if an address is registered as an AI agent under ERC-8004.

## Demo

Open `erc8004-lookup.html` in a browser.

## Usage

```bash
# Run demo script
bash scripts/erc8004_lookup_demo.sh
```

## What It Does

- Takes a Base/Ethereum address
- Checks if registered as ERC-8004 agent
- Shows agent metadata if found

## Files

- `erc8004-lookup.html` - Web interface
- `scripts/erc8004_lookup_demo.sh` - Demo script
